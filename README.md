3.1 Público Objetivo

El proyecto web está orientado a un público joven con interés en plataformas digitales de entretenimiento y contenido interactivo.

Rango etario: 18 a 35 años, enfocado principalmente en adolescentes y adultos jóvenes.

Nacionalidad: Principalmente usuarios de Latinoamérica.

Ubicación geográfica: Mayormente en contextos urbanos o semiurbanos, donde el acceso a internet es frecuente y estable.

Variables adicionales relevantes:

Nivel educativo: Secundario completo o en curso, y estudiantes universitarios.

Ocupación: Estudiantes y jóvenes trabajadores.

Nivel de alfabetización digital: Medio a alto, familiarizados con plataformas web y aplicaciones.

Dispositivos de acceso: Principalmente teléfonos móviles, aunque también computadoras.

Idioma principal: Español.

Este público fue elegido porque representa una gran parte de los usuarios activos en plataformas digitales modernas, permitiendo un diseño enfocado en la simplicidad, rapidez y estética atractiva sin dejar de ser funcional.

3.2 Design Inspirations and Creative Process

Design Inspirations
The design of this web application is inspired by platforms such as Game Jolt, which stand out for their clean interface and user-friendly navigation. The goal is to achieve a balance between simplicity and visual appeal.

Game Jolt: Inspires the structure and community-oriented layout, with a focus on accessibility and content discovery.

Minimalist design trends: Influence the use of clean layouts, reduced visual noise, and clear hierarchy.

Modern UI styles: Use of soft colors, simple typography, and intuitive components to improve user experience.
These inspirations were chosen to create a platform that feels modern, accessible, and engaging without overwhelming the user.

Creative Process Documentation

The creative process started with the idea of building a simple yet visually appealing platform. The main focus was to prioritize usability while maintaining an attractive design.

At first, more complex interface ideas were considered, including multiple sections and advanced visual effects. However, these were discarded in favor of a cleaner and more intuitive design, ensuring better performance and ease of use.

The final concept evolved into a minimalist interface with clear navigation, inspired by existing successful platforms but adapted to the needs of the target audience. Design decisions were guided by the intention to make the website easy to use, visually pleasant, and accessible from different devices.
Overall, the project reflects a balance between inspiration and originality, focusing on delivering a smooth and enjoyable user experience.
